# Генеалогическое дерево семьи
Консольная программа строит генеалогическое дерево семьи из файла и показывает родословную в разном виде. Дерево двоичное, в каждом узле один человек, слева его отец, справа его мать. Программа выводит родословную тремя обходами, находит всех предков выбранного человека, сортирует семью по дате рождения и умеет обходить дерево через стек без рекурсии.

## Что делает программа
- Загружает семью из файла JSON с именами, родителями и датами рождения
- Строит двоичное дерево предков, где у человека отец слева и мать справа
- Показывает родословную тремя обходами, прямым, обратным и симметричным
- Находит всех предков выбранного человека
- Сортирует членов семьи по дате рождения
- Выводит обход дерева в глубину через стек без рекурсии
- Работает через простое меню в консоли

## Файлы проекта
- `tree.py` содержит класс `Node`, построение дерева, три рекурсивных обхода, поиск узла, поиск предков, сортировку и обход через стек
- `loader.py` читает данные о семье из файла JSON
- `main.py` содержит заголовок, меню, вывод списков и главный цикл
- `family.json` хранит пример семьи на три поколения

## Как устроена программа
Словарь `people` связывает имя человека с его записью из файла. Функция `build_tree` работает рекурсивно, она создает узел для человека, затем строит слева дерево его отца, а справа дерево его матери, пустое имя завершает ветку. Так как у каждого человека ровно два родителя, дерево получается двоичным. Обходы спускаются по дереву рекурсивно и собирают узлы в список. Поиск предков сначала рекурсивно находит нужный узел, а потом собирает всех людей из его поддерева отца и матери. Сортировка и обход через стек работают со списком узлов, который заранее собран из дерева.

## Структуры данных и алгоритмы
- Двоичное дерево хранит родственные связи, у каждого узла есть отец и мать
- Рекурсия применяется при построении дерева, трех обходах и поиске узла
- Прямой, обратный и симметричный обходы выводят родословную в разном порядке
- Стек на основе списка с `append` и `pop` дает обход в глубину без рекурсии
- Сортировка выбором упорядочивает членов семьи по дате рождения
- Рекурсивный поиск по дереву находит человека и его предков

## Как запустить
Нужен установленный Python 3. Положите все файлы рядом в одну папку и запустите команду `python main.py` из этой папки. Программа сама прочитает `family.json` и покажет меню.
