# Генеалогическое дерево семьи

Консольное приложение, которое собирает семейное дерево из файла и показывает родословную с разных сторон. Дерево двоичное, узел это человек, слева отец, справа мать.

## Запуск
Нужен только Python 3. Положите файлы в одну папку и запустите `python main.py`. Данные возьмутся из `family.json`, после чего откроется меню.

## Пример работы
```
ГЕНЕАЛОГИЧЕСКОЕ ДЕРЕВО
  1. Родословная, прямой обход
  2. Родословная, обратный обход
  3. Родословная, симметричный обход
  4. Найти предков человека
  5. Члены семьи по дате рождения
  6. Обход в глубину через стек
  0. Выход
Ваш выбор: 4
Имя человека: Дмитрий
Предки человека "Дмитрий":
  1. Сергей, 1949-01-10
  2. Иван, 1923-02-14
  3. Григорий, 1898-09-12
  4. Тамара, 1901-12-25
  5. Ольга, 1925-08-08
  6. Николай, 1900-06-19
  7. Лидия, 1951-05-30
  8. Зоя, 1927-10-08
```

## Пункты меню
Пункт | Что делает
--- | ---
Прямой обход | показывает родословную в порядке человек, отец, мать
Обратный обход | показывает сначала предков, потом самого человека
Симметричный обход | показывает отца, человека, мать
Найти предков | выводит всех предков выбранного человека
По дате рождения | сортирует всю семью по возрастанию даты
Обход через стек | проходит дерево в глубину без рекурсии

## Формат данных
Файл `family.json` это список людей. У каждого человека есть имя, отец, мать и дата рождения. Если поле отца или матери пустое, значит этот родитель неизвестен и ветка в эту сторону не продолжается. У кого-то известны оба родителя, у кого-то только один, а у самых дальних предков их нет совсем. Корень дерева задается `ROOT_NAME` в `main.py`.
```json
{"name": "Анна", "father": "Дмитрий", "mother": "Елена", "birth": "2000-03-15"}
```

## Структуры данных и алгоритмы
Что используется | За что отвечает
--- | ---
Двоичное дерево | хранит родственные связи, у узла есть отец и мать
Рекурсия | построение дерева, три обхода, поиск узла
Стек на списке | обход в глубину без рекурсии
Сортировка выбором | упорядочивает семью по дате рождения
Рекурсивный поиск | находит человека и его предков

## Устройство проекта
- `tree.py` содержит класс `Node`, построение дерева, три обхода, поиск узла, поиск предков, сортировку и обход через стек
- `loader.py` читает данные о семье из файла JSON
- `main.py` содержит заголовок, меню, вывод списков и главный цикл
- `family.json` хранит пример семьи

Словарь `people` связывает имя человека с его записью из файла. Функция `build_tree` работает рекурсивно. Она создает узел, строит слева дерево отца, справа дерево матери, а пустое имя завершает ветку. У каждого узла не больше двух потомков, отец и мать, поэтому дерево двоичное. Поиск предков сначала рекурсивно находит нужный узел, а затем собирает всех людей из его поддерева. Список членов семьи для сортировки собирается из всех записей файла, а обход в глубину идет прямо по дереву через стек.
